# Brain — Index & Usage Guide

This is your AI's shared memory. Every file here is readable by Claude Code.

## Structure

| Folder/File | Purpose |
|-------------|---------|
| `goals.md` | Vision → yearly → quarterly → weekly priorities |
| `projects/` | Active project dashboard |
| `people/` | Contact CRM |
| `sops/` | Standard operating procedures |
| `daily/` | Daily logs |
| `calendar/` | Annual planning |
| `knowledge-base/` | Domain expertise files |
| `research/` | Research output |
| `pipeline/` | Deal pipeline |
| `drafts/` | Draft content |
| `invoices/` | Invoice records |

## How to Use

- Add files anywhere in this folder — Claude will find them
- Reference files in CLAUDE.md with @brain/filename.md
- Ask Claude to "save this to my brain" and it will file it correctly
